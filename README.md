# Cents 💸

> **Minimal expense management. Every cent accounted for.**

A beautifully minimal, mobile-first budget tracker built as a single-file web app — no install, no build step, works offline.

## Live App

🔗 **[Open on GitHub Pages](https://YOUR_USERNAME.github.io/cents)**

Scan this on your phone — it's optimized for mobile.

## Features

- 📊 **Budget ring** — animated SVG showing monthly spend at a glance
- 🏷️ **Categories** — set per-category limits with min & max alert thresholds
- 🔔 **Smart alerts** — toast notifications when you approach or hit limits
- 📈 **Analytics** — donut chart (by category) + bar chart (daily trend)
- 📋 **History** — filterable expense log, grouped by day
- 💾 **JSON import/export** — full data portability between devices
- 🌍 **Multi-currency** — 9 currency symbols supported
- 💿 **Offline-first** — all data stored in browser localStorage

## Tech Stack

- React 18 (via CDN)
- Chart.js 4
- Vanilla CSS design system
- Zero dependencies, zero build step

## Design

Pure minimal light grey theme — black, white, and grey only. Designed like an iOS app in the browser.
